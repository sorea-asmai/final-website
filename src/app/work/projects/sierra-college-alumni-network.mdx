## Overview

This project involved collaboration with the Sierra College Alumni Foundation to develop a web application that enhances alumni engagement and reflects the college's branding. The focus was on creating a user-friendly platform that fosters connections among alumni and facilitates donations.

## Key Features

- **User Profiles**: Customizable profiles were implemented for alumni, allowing them to showcase their achievements and connect with peers.
- **Donation System**: A secure and straightforward donation page was developed to encourage contributions to the foundation.
- **Responsive Design**: The application was designed to be mobile-friendly, providing a seamless experience across devices.
- **Collaboration Tools**: Features for event planning and communication among alumni were integrated.

## Technologies Used
- **Figma**: For designing UI components and prototypes.
- **React**: For building a dynamic and interactive user interface.
- **JavaScript**: For adding interactivity and managing application logic.
- **Sass**: For modular and maintainable styling.


## Challenges and Learnings

A significant challenge was ensuring the application met the varying needs of alumni while maintaining a cohesive design. Frequent meetings with the client were essential for gathering feedback and aligning on requirements. Collaborating with a diverse team of seven provided valuable insights into effective communication and project management.

## Contributions

In addition to core development tasks, the project also involved key contributions in the following areas:

- **Redesign & Prototyping**: Redesigned and prototyped screens in Figma to align with the style guide, prioritizing a seamless user experience.
- **Front-End Development**: Translated mockups into a functional, visually appealing interface, enhancing front-end coding skills in React, JavaScript, and SCSS.
- **Comprehensive Web Solution**: Worked with a team of seven to develop a comprehensive solution integrating alumni networking, donation functionality, verification applications, and search features, all aimed at enhancing alumni engagement.

## Outcome

The web application successfully reflects the college's branding while fostering connections among alumni and enhancing engagement. It has become a vital tool for alumni to stay connected, share opportunities, and support the foundation through donations.
